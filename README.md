# 一个随意的加密工具集

欢迎来到这个充满随性的加密工具仓库！这里汇集了一些简单、实用且“不怎么讲究”的加密脚本，主要是为了方便日常快速使用，而不是为了构建严肃的安全系统。

## 简介

本仓库包含了几种借助 **OpenSSL** 或其他常用工具编写的加密脚本，语言涵盖 Batch、PowerShell 和 Python。这些脚本不追求代码的精美与严谨，只求能快速完成以下目标：

- 对文本或文件进行简单加密/解密
- 使用对称或非对称加密算法
- 支持 Base64、密钥文件等常见交互方式
- 一键生成密钥对、导入导出密钥
- 友好的交互式命令行界面

## 支持的算法与功能

目前已经（或即将）支持以下加密方式：

- **RSA**（4096位密钥）
- 密钥管理与导出

你可以使用这些脚本进行文本加密、文件加密、密钥交换、数字签名（如果我想起来实现的话）等操作。

## 使用场景

这些脚本适用于：

- 快速加密一小段敏感信息
- 在本地环境中安全传递文件
- 学习或演示加密过程
- 作为其他项目的辅助工具
- 临时需要加密但不想安装大型软件的情况

## 如何使用

1. 确保已安装 **OpenSSL**（Windows 用户可以从[官方网站](https://slproweb.com/products/Win32OpenSSL.html)下载安装）
2. 克隆或下载本仓库到本地，又或是下载发行版中的内容
3. 运行对应语言的脚本（BAT、PS1 或 PY 文件）
4. 按照提示选择操作（加密、解密、生成密钥等）

每个脚本都带有简单的交互菜单，无需记忆复杂命令。

## 为什么叫“随意的加密工具”

因为这里的代码可能：

- 基于GPT的编程
- 没有完整的错误处理
- 缺少详细的注释
- 使用了不够优雅的实现方式
- 随时可能被我修改或替换

但它们都能 **正常工作**，并且足够简单到你可以一眼看懂它在做什么（或者至少能猜到）。

## 注意事项

- **不要**将这些脚本用于真正高安全需求的场景
- 生成的密钥文件请妥善保管，丢失无法恢复
- 加密强度取决于密钥长度与算法，请根据实际情况选择
- 本仓库不保证长期维护，脚本可能随时更新或删除
